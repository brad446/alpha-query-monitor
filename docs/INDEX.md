## Usage Guidelines

1. **Installation**: Clone the repository using:
   ```bash
   git clone https://github.com/brad446/alpha-query-monitor.git
   ```
2. **Setup**: Navigate into the project directory and install the required dependencies:
   ```bash
   cd alpha-query-monitor
   npm install
   ```
3. **Running the Application**: Start the application with:
   ```bash
   npm start
   ```
4. **Contributing**: Any contributions are welcomed! Please ensure to follow the guidelines set in `guidelines.md`.
